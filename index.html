Diseño y Aplicación de Bases de Datos

Tiene como propósito formar a los estudiantes en el desarrollo de aplicaciones web completas, integrando la creación de interfaces modernas con HTML, CSS, JavaScript en el frontend, y la gestión eficiente de la información mediante MongoDB en el backend. A lo largo del curso, se abordarán desde los fundamentos de bases de datos y programación web hasta la integración y despliegue de proyectos, con un enfoque práctico y orientado a resolver problemas reales del ámbito profesional.

Contenido Teórico

  • Estructura general: frontend (HTML, CSS, JS, Bootstrap) y backend (MongoDB).
  • Metodología: teoría + práctica + proyectos.
  • Herramientas a utilizar:
    • Frontend: Visual Studio Code, navegadores modernos.
    • Backend: MongoDB, Node.js (Express más adelante).

Definición: Una base de datos es una colección organizada de información que puede ser accedida, gestionada y actualizada.

Evolución: Archivos planos → Bases de datos relacionales → Bases de datos NoSQL.

Tipos de bases de datos más comunes:

  • Relacionales (SQL): MySQL, PostgreSQL, SQL Server.
  • NoSQL: MongoDB, Firebase, Cassandra.

Relacional (SQL):

  • Organizada en tablas.
  • Usa lenguaje SQL.
  • Estructura rígida (esquemas).
  • Ejemplo: sistema bancario.

NoSQL (MongoDB):

  • Organizada en documentos JSON.
  • Flexible y escalable.
  • Ideal para aplicaciones web modernas.
  • Ejemplo: redes sociales, e-commerce.

  • Frontend: Interfaz visible para el usuario (HTML, CSS, JS, Bootstrap).
  • Backend: Lógica de negocio y conexión con la base de datos (Node.js, Express, MongoDB).
  • Base de datos: Almacenamiento persistente de la información.

Actividad práctica

Actividad: Elaborar un mapa conceptual digital (Coggle, Canva, Miro o PowerPoint) con los siguientes puntos:

  1. Tipos de bases de datos.
  2. Diferencias entre SQL y NoSQL.
  3. Ubicación del frontend, backend y base de datos dentro de una aplicación web.

Entrega: PDF o captura del mapa conceptual.

Lectura recomendada

  • Duckett, J. (2014). HTML and CSS: Design and Build Websites. Wiley. (Introducción a la web, pp. 12–25).
